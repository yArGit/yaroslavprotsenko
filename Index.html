  • git config --global user.name "Ім'я" Вказує Ваше "Ім'я" для відображення у коммітах
  • git config --global user.email "електронна пошта" Вказує Вашу пошту для відображення у коммітах
  • git init Cтворює віртуальну папку для роботи в GIT
  • git status Виводить стан файлів(Змінені/не змінені/очікують додавання в індекс) у робочій директорії
  • git add Додає файл у тимчасову папку для відслідковування та комміту
  • git commit зберегає всі дані, які були додані в індекс за допомогою git add
  • git log Виводить історію коммітів
  • git checkout Використовується для перемикання гілок
  • git branch Виводить гілки проекту. git branch 'назва' створює нову гілку
  • git push Зберігає дані у віддалений репозиторій
  • git pull Записує дані віддаленого репозиторію у нашу гілку
  • git merge 'назва' Зберігає зміни гілки 'назва' у гілку мастер
  • git clone Клонує віддалений репозиторій у нову дикректорію зі збереженням гілок відслідковування
  • git revert Відміняє зміни, які були внесенні, одним за попередніх, коммітом
  • git reset Відміняє зміни. git reset -hard відміняє зміни до останнього комміту. git reset -soft відміняє останній комміт
  • git rebase є автоматизованим git cherry-pick, що застосовуэться для декількох коммітів
  • git cherry-pick Використовується для внесення змін, які вже були внесені іншим коммітом. Створює новий комміт
  • git stash Зберігає всі зміни без комміту
  • .gitignore файл використовується для того, щоб незадіяні файли проекту ігнорувались git. Можна викорустовувати шаблони.
